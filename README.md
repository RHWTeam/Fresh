# Fresh  
A simple and beautiful WordPress theme.  

# DEMO  
[Rock's Blog](https://rockethcgs.me/)  

# 特性  
- 使用Bootstrap框架开发  
- 响应式界面，双栏  
- 极简主义、清爽、没有过多设置  
- 自定义meta，底部版权信息  
- 内置访问统计、面包屑导航  
- 炫(la)酷(ji)的特效  
- 免费
