# Sunshine and coffee  
A simple and beautiful WordPress theme.  

# DEMO  
[RHW Home](https://home.rhw-team.com/)  
  
![img](https://home.rhw-team.com/wp-content/uploads/theme-demo.png)  

# 特性  
- 使用Bootstrap框架开发  
- 响应式界面，双栏  
- 极简主义、清爽、没有过多设置  
- 自定义meta，底部版权信息  
- 内置访问统计、面包屑导航  
- 炫(la)酷(ji)的特效  
- 免费
